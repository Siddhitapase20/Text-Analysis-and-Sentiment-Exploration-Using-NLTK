# Text Analysis and Sentiment Exploration Using NLTK

## Overview
This project performs text preprocessing, frequency analysis, named entity recognition, and sentiment analysis using Python and the NLTK library.

## Features
- Sentence tokenization
- Word tokenization
- Stopword removal
- Lemmatization
- POS tagging
- Named Entity Recognition (NER)
- Sentiment Analysis using VADER
- Frequency distribution visualization

## Technologies Used
- Python
- NLTK
- Matplotlib

## Dataset
Shakespeare Hamlet dataset from the NLTK Gutenberg Corpus.

## Output
The project generates:
- Top frequent words
- Most frequent nouns
- Named entities
- Sentiment distribution pie chart
